L'outil d'impression des étiquettes à MaxiBazar

Genre : Projet Stage (BTS - 2eme année)
Date : 2024

Langages : Python
Outils et frameworks : PyCharm; Tkinter

Cet outil est conçu pour générer un document PDF contenant des étiquettes de rayonnage à partir d'un fichier Excel. Il est capable d'imprimer les étiquettes de tous les produits répertoriés dans le fichier fourni. Les utilisateurs peuvent configurer les dimensions du document, ainsi que le type de code-barres et d'autres options.
